== Description ==

This plugin supports you to receive earthquake information and weather forecasts in Jakarta and its surroundings, maybe you will travel and need this information.

This data is taken from BMKG Indonesia data.

BMKG is very big for disaster mitigation in Indonesia. Because of information from BMKG, the public can increase their vigilance and caution in responding to disaster phenomena.

This plugin is very easy to install, even if you are not a programmer

== Installation ==

1. Upload `prakiraan-cuaca-dan-info-gempa.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a widget or web page and then insert the sort code [prakiraan-cuaca]
